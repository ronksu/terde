terassibongari
=====

### Dev instructions:
  1. install node & npm: http://nodejs.org/
  2. install sass/compass: http://thesassway.com/beginner/getting-started-with-sass-and-compass
  3. clone repo 
  4. run `npm install`
  5. run `bower install`
  6. run `grunt server`


Yeoman might be nice to get the dependencies right: http://yeoman.io/
